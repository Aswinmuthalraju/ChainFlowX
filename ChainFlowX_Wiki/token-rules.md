# Token Optimization Rules

> Read ONLY when a CLAUDE.md rule is unclear. NOT on every session.

1. index.md first — always. It's the map; blind scanning wastes tokens.
2. Max 3 wiki files in context at once. Exceptions: LINT, SCAN.
3. Write tight. Cross-link; never duplicate content across pages.
4. Sources: key takeaways only, max 20 lines per source page.
5. Touch only pages affected by the current operation.
6. Log/index entries: one line, no sub-bullets, no prose.
7. LINT is read-only. Checklist only. No auto-fixes.
8. SESSION END: delta writes only. Skip if nothing changed.
9. Code is truth. If wiki disagrees with code, update wiki.
