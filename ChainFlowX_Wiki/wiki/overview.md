# ChainFlowX — Project Overview
_Last updated: 2026-04-09 | Sources: 1_

## Project
- **Name**: ChainFlowX
- **Version**: v5.1 (Stability Hardened)
- **Description**: Supply chain contagion intelligence engine — models disruption shock waves through a dependency network with explainable AI, Disruption DNA™ pattern matching, Ripple Score™ propagation, and dual local AI models on a production-grade 3D globe
- **Stack**: React 18 + Vite + JavaScript, globe.gl + Three.js, deck.gl + MapLibre, Tailwind CSS, Gemma 4 E4B + Qwen3:8B (Ollama/ngrok), GDELT API, Vercel
- **Hackathon**: April 9–10, 2026 | Built solo by Aswin

## Synthesis
ChainFlowX is a 6-layer intelligence stack forked on top of WorldMonitor (44.1k stars). It addresses two gaps in the $4.2T supply chain risk market: (1) black-box risk scores with no derivation, and (2) reaction dashboards with no propagation modeling. The system treats disruptions as contagion waves — quantified by Ripple Score™, matched against historical fingerprints via Disruption DNA™, with downstream industry impact gated by actual cascade severity. All six layers run offline at $0 API cost using two local AI models on dedicated hardware.
